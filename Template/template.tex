	\h{Rank of a Matrix}
	
	Suppose $A$ is a matrix with RREF $R$. The \textbf{rank} of $A$ is the number of leading ones in $R$.
	
	\h{Theorem 2.2.4}
	
	If $A$ is a $m\times n$ matrix, then Rank $A \leq \min(m, n)$
